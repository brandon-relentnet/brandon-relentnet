# Brandon Harris

I build polished applications, useful automation, and client-ready websites with a strong bias toward clean interfaces and practical workflows.

I run [RelentNet](https://relentnet.com), a white-glove software and web development studio for small businesses. I also build and operate [myscrollr](https://myscrollr.com), a browser extension for pinning live sports scores, crypto prices, and custom feeds over any tab.

## What I Build

- Polished web apps with strong visual direction
- Automation that removes repetitive work
- Product experiments that solve real annoyances
